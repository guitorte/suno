## 1. Meta Tags Fundamentais

### O que são Meta Tags

**Meta tags** são palavras-chave que orientam o SunoAI na criação musical. Funcionam como "instruções criativas" que definem gênero, humor, instrumentos, efeitos e estrutura da música.

### Como Funcionam

| Contexto | Formato | Exemplo |
|----------|---------|---------|
| **Drag & Drop (Letra)** | Com colchetes `[Tag]` | `[Verse]`, `[Chorus]` |
| **Campo Style** | Sem colchetes | `Pop, Emotional, Piano` |

### 4.1 Controle de Registro Vocal por Seção

**Registros Disponíveis:**

| Registro | Característica | Melhor Para |
|----------|----------------|-------------|
| `Falsetto` | Leve, arejado, agudo | Momentos etéreos, vulnerabilidade |
| `Chest Voice` | Cheio, poderoso, fundamentado | Entrega confiante |
| `Head Voice` | Claro, leve, agudo mas cheio | Brilho pop |
| `Mixed Voice` | Balanceado, versátil | Pop/R&B moderno |
| `Whispered` | Suave, íntimo | Segredos, vulnerabilidade |
| `Spoken Word` | Narrativo, não-melódico | Versos de rap, intros |
| `Raspy` | Rouco, granuloso | Rock, blues, emoção crua |
| `Belted` | Potência máxima, volume alto | Momentos de clímax |

**Mapeamento Emocional:**

| Registro | Emoção |
|----------|--------|
| Falsetto | Vulnerabilidade, sonho, desejo |
| Chest | Força, determinação, confiança |
| Whispered | Intimidade, segredos, fragilidade |
| Belted | Clímax, libertação, triunfo |
| Raspy | Dor, experiência, autenticidade |

**Quando Falha:**
- ❌ Tempo muito rápido (sem tempo para mudanças)
- ❌ Letras muito densas
- ❌ Muitas mudanças de registro (mais de 4-5)
- ❌ Registradores contradizem o gênero

**Conselhos de Iteração:**
- Use em momentos estruturais — verso vs refrão, não cada linha
- Crie arco — comece suave, construa para poderoso
- Combine emoção com registro — vulnerável = sussurrado/falsete
- Não exagere — 3-4 mudanças de registro por música

---

### 4.2 Troca de Emoção por Seção

**Como a IA Responde a Tags de Emoção:**

| Tag Emocional | Ajuste da IA |
|---------------|--------------|
| Sad | Tom menor, mais lento, esparso |
| Angry | Voz agressiva, mais rápido, mais alto |
| Desperate | Instrumentos esparsos, vocais crus |
| Explosive | Banda completa, energia máxima |
| Melancholic | Tempo lento, harmonias menores |
| Euphoric | Mais rápido, tom maior, brilhante |
| Defiant | Entrega forte, poderosa |
| Reflective | Suave, ritmo pensativo |
| Triumphant | Arranjo completo, vitorioso |

**Quando Falha:**
- ❌ Emoções contradizem tempo (triste + rápido)
- ❌ Tudo é rotulado como emocional (sem contraste)
- ❌ Emoções mudam frequentemente demais
- ❌ Labels são muito sutis ou similares

**Insight Principal:** Emoção só funciona quando **muda**. Contraste é tudo.

**Conselhos de Iteração:**
- Crie arco emocional claro — começo → meio → fim
- Use emoções contrastantes — verso triste, refrão triunfante
- Limite labels de emoção — 3-4 por música
- Combine com letra — emoção deve reforçar as palavras

### 4.3 Adlibs Vocais com Colchetes

**Sintaxe:**
```
I'm on top [yeah!] of the world [uh!]
Living my life [let's go!] no regrets [ayy!]
```

**Adlibs por Gênero:**

| Gênero | Adlibs Comuns |
|--------|---------------|
| Hip Hop/Trap | [yeah!], [uh!], [ayy!], [skrrt!], [woah!], [let's go!] |
| R&B/Soul | [ooh], [ahhh], [mmm], [oh yeah], [baby] |
| Rock/Metal | [hey!], [go!], [yeah!], [whoa-oh], [come on!] |
| Pop | [oh oh oh], [na na na], [la la la], [hey hey] |
| Latin | [dale!], [oye!], [vamos!] |

**Quando Falha:**
- ❌ Usado com muita frequência (cada palavra)
- ❌ Letras já estão sobrecarregadas
- ❌ Gênero não usa adlibs (clássico, ambiente)
- ❌ Adlibs não combinam com convenções do gênero

**Conselhos de Iteração:**
- Adlibs = pontuação, não frases completas
- Combine com gênero — use adlibs apropriados
- Não sobrecarregue — 2-3 por verso é suficiente
- Coloque estrategicamente — fim de linhas, entre frases

### 4.4 Parênteses para Vocais de Fundo

**Como Funciona:**
- Texto em `(parênteses)` é tratado como secundário, fundo, suporte
- Suno renderiza como backing vocals, harmonias, call-and-response, adlibs

**Exemplos por Gênero:**

**Call-and-Response (Hip Hop/R&B):**
```
I'm walking alone (walking alone)
Through the city lights (ooh ooh ooh)
Nobody knows my name (knows my name)
```

**Gospel/Soul Harmonies:**
```
He lifted me up (lifted me up!)
Out of the darkness (hallelujah!)
Into the light (praise the Lord!)
```

**Trap Adlibs:**
```
I got that money (yeah!)
Stacking it high (skrrt skrrt!)
Living my life (ayy! ayy!)
```

**Camadas Múltiplas:**
```
I'm breaking free (breaking free) (ooh) (yeah yeah)
```

**Conselhos de Iteração:**
- Use consistentemente — estabeleça o padrão cedo
- Varie o conteúdo — misture ecos, adlibs, harmonias
- Não sobrecarregue — máximo 1-2 parênteses por linha
- Combine com convenções do gênero

### 4.5 Forçar Dueto Multi-Voz

**Tags Vocais Disponíveis:**

| Tag | Função |
|-----|--------|
| `[Male Vocal]` | Voz masculina |
| `[Female Vocal]` | Voz feminina |
| `[Both Vocals]` | Ambos juntos |
| `[Male Lead, Female Backing]` | Masculino frente, feminino fundo |
| `[Female Lead, Male Harmony]` | Feminino frente, masculino harmonia |
| `[Duet]` | Ambas vozes, balanceadas |

**Requisito Crítico:** Você DEVE especificar "Duet" no prompt Style!

**Quando Falha:**
- ❌ "Duet" ausente do Style (crítico!)
- ❌ Ambos papéis têm letras idênticas
- ❌ Labels de seção são confusos
- ❌ Muitas trocas de voz (confunde a IA)

**Conselhos de Iteração:**
- Sempre inclua "Duet" no style — não negociável
- Labels claros de seção — seja explícito sobre quem canta
- Contraste as letras — perspectivas diferentes ajudam diferenciação
- Limite trocas de voz — máximo 4-6 por música

### 4.6 All Caps para Picos Emocionais

**Como Funciona:**
- Letras maiúsculas sinalizam ênfase, urgência, intensidade
- Suno interpreta ALL CAPS como "Aumente energia vocal"

**Sintaxe:**

| Formato | Efeito |
|---------|--------|
| `ALL CAPS + !` | Poderoso, gritado |
| `ALL CAPS + ?` | Dramático, desesperado |
| `ALL CAPS + ?!` | Questionamento intenso |
| `Mixed case + !` | Ênfase normal |

**Quando Falha:**
- ❌ Tudo em caps (sem contraste)
- ❌ Gênero desencoraja vocais fortes (ambient, minimal)

**Conselhos de Iteração:**
- Use caps apenas para MOMENTOS DE CLÍMAX
- Crie contraste — verso quieto, refrão alto
- Adicione pontuação — ! e ?! reforçam intensidade
- Não exagere — se tudo grita, nada grita

### 4.7 Vocal Stretching e Stuttering

**Técnica #1: Stretching (Notas Longas)**
```
I lo-o-o-o-o-ve you
Forever-e-e-e-r-r-r
Sta-a-a-a-ay with me-e-e-e
```
- Cada hífen = duração estendida de nota
- Melhor para: Baladas, soul, gospel, momentos emocionais

**Técnica #2: Stutter Effect**
```
I-I-I-I want want want you you you
B-b-b-baby baby b-b-baby
```
- Repetições = gagueira estilo autotune
- Melhor para: Trap, hip hop, electronic pop, EDM

**Técnica #3: Melodic Spelling**
```
L-O-V-E spells love
G-O-O-D-B-Y-E means the end
```
- Cada letra vira nota separada
- Melhor para: Pop hooks, jingles, músicas alegres

**Quando Falha:**
- ❌ Usado durante toda a música
- ❌ Aplicado a tempos muito rápidos
- ❌ Empilhado com letras densas
- ❌ Gênero não suporta vocais estendidos

**Conselhos de Iteração:**
- Use com moderação — apenas palavras/frases-chave
- Combine com tempo — stretching funciona melhor em tempos lentos
- Considere gênero — trap adora stuttering, folk não
- Teste com 2-3 variações antes de comprometer

## 5. Técnicas de Estrutura e Letras

### 5.1 Tag Stacking com PIPE

**Problema (SEM PIPE):**
```
❌ ERRADO (9 adjetivos, IA confusa):
[Chorus: Anthemic, Powerful, Epic, Orchestral, Dramatic,
Climactic, Intense, Explosive, Grandiose]
```

**Solução (COM PIPE):**
```
✅ CORRETO (7 elementos, claramente separados):
[Chorus | Anthemic | Stacked Harmonies | Brass Section |
Bass Drop | Wide Stereo | Heavy Compression]
```

**Ordem Ótima de Elementos:**
```
[Section | Mood/Energy | Vocal Style | Key Instruments |
Dynamic/Movement | Spatial/Effects | Production Style]
```

**Exemplos por Gênero:**

**EDM Drop:**
```
[Drop | Explosive | Vocoder Vocals | Synth Lead |
Bass Drop | Sidechain Compression | Wall of Sound]
```

**Jazz Ballad:**
```
[Verse | Intimate | Smooth Female Vocals | Piano Solo |
Gentle Dynamics | Close Mic | Warm Analog]
```

**Rock Anthem:**
```
[Chorus | Anthemic | Full Band | Distorted Guitars |
Building Energy | Wide Stereo | Arena Sound]
```

**Quando Falha:**
- ❌ Mais de 7 elementos por tag
- ❌ Elementos contraditórios (íntimo + explosivo juntos)
- ❌ Usado sem estrutura clara de seção

**Conselhos de Iteração:**
- MÁXIMO 7 elementos por tag — além disso = confusão
- Priorize — coloque elementos mais importantes primeiro
- Seja específico — "Guitar Solo" não apenas "Guitar"
- Combine com prompt de style — tags devem reforçar, não contradizer

### 5.2 Concerto ao Vivo com Efeitos Sonoros

**Sintaxe de Seção:**
```
[Intro: Live Crowd Cheering]
[Stage Ambience]
[Outro: Applause, Crowd Going Wild]
```

**Sintaxe de Asterisco (sons não cantados):**
```
*crowd roaring*
*audience cheering*
*applause*
```

**Referência de Efeitos Sonoros:**

**Atmosferas ao Vivo:**
| Sintaxe | Efeito |
|---------|--------|
| `*crowd cheering*` | Aplausos e gritos |
| `*audience singalong*` | Público cantando junto |
| `*festival roar*` | Rugido de festival/estádio |
| `*applause*` | Aplausos finais |

**Efeitos Dramáticos:**
| Sintaxe | Efeito |
|---------|--------|
| `*thunder*` | Trovão para momentos épicos |
| `*gunshots*` | Para rap/trap/metal |
| `*explosion*` | Para drops de EDM |
| `*glass breaking*` | Vidro quebrando |

**Ambientes:**
| Sintaxe | Efeito |
|---------|--------|
| `*café ambience*` | Conversas de cafeteria |
| `*rain falling*` | Atmosfera melancólica |
| `*wind howling*` | Sensação isolada, fria |

**Narrativa:**
| Sintaxe | Efeito |
|---------|--------|
| `*phone ringing*` | Telefone tocando |
| `*door slamming*` | Porta batendo |
| `*footsteps*` | Passos |
| `*heartbeat*` | Pulso de batimento cardíaco |

**Quando Falha:**
- ❌ Usado com gêneros altamente polidos (pop limpo, EDM)
- ❌ Combinado com keywords "clean / studio / pristine"
- ❌ Exagerado (muitos cues de efeito sonoro)
- ❌ Cues contradizem o prompt de style

**Conselhos de Iteração:**
- Menos é mais — um ou dois cues ao vivo são suficientes
- Combine com gênero — cues ao vivo funcionam melhor com rock, folk, singer-songwriter
- Coloque cues em momentos estruturais — intro, entre seções, outro
- Não espere efeitos sonoros precisos — pense "atmosfera", não "som específico"

### 5.3 Reescrita Fonética para Controle de Pronúncia

**Quando Usar:**
- ✅ Palavra consistentemente mal pronunciada em várias gerações
- ✅ Homógrafos (mesma grafia, som diferente) são lidos errado
- ✅ Termos técnicos ou nomes são deturpados
- ✅ Precisa de pronúncia precisa para trocadilho ou rima

**Técnica #1: Reescrita Fonética Simples**

| Grafia Padrão | Reescrita Fonética | Por quê |
|---------------|---------------------|---------|
| **read** (presente) | `reed` | Força "ree-d" em vez de "red" |
| **live** (concerto) | `lyve` | Força "laiv" em vez de "liv" |
| **bass** (instrumento) | `bahss` ou `basss` | Evita pronúncia "base" |
| **tear** (choro) | `teer` | Força "teer" em vez de "tare" |
| **wound** (ferida) | `woond` | Força "woond" em vez de "wownd" |
| **lead** (chumbo) | `led` | Força "led" em vez de "leed" |

**Técnica #2: Separação Silábica com Hífens**
```
extraordinary → ex-traor-din-ary
catastrophe → ca-tas-tro-phe
pneumonia → new-moan-ya
```

**Técnica #3: IPA para Palavras Teimosas**

Use IPA quando:
- ❌ Reescrita fonética ainda falha
- ❌ Palavra é altamente incomum ou técnica
- ❌ Precisa de precisão cirúrgica para uma palavra

**Exemplo IPA:** `"breath"` vs `"breathe"`
```
I'm out of /brɛθ/ again
```

**Quando Falha:**
- ❌ Grafia fonética cria NOVA pronúncia errada
- ❌ IPA usado para muitas palavras (confunde modelo)
- ❌ Reescrita é muito diferente da palavra original
- ❌ Suno interpreta reescrita como palavra completamente diferente

**Conselhos de Iteração:**
1. Comece simples — tente reescrita fonética básica primeiro
2. Adicione hífens se necessário — divida palavras teimosas
3. Reserve IPA para último recurso — apenas 1-2 palavras problema máximo
4. Teste incrementalmente — corrija uma palavra por vez
5. Não exagere — se a letra fica ilegível para humanos, não funcionará

### 5.4 Equipment Keywords de Estúdio Profissional

**⚠️ Aviso:** O AI NÃO simula equipamento real. Estas keywords funcionam (se funcionarem) por associação semântica.

**Microfones:**
```
"Neumann U 87 microphone, pristine vocal capture"
"SM58, live raw sound"
"Ribbon microphone, warm vintage tone"
```

**Preamps e Consoles:**
```
"Neve console warmth, API punch"
"Mixed on SSL, clean and powerful"
"Tube preamp saturation"
```

**Meio de Gravação:**
```
"Recorded to tape, analog warmth"
"2-inch tape machine, vintage saturation"
```

**Quando Falha:**
- ❌ Empilhado excessivamente (retornos decrescentes)
- ❌ Usado como substituto de boa estrutura
- ❌ Esperado para "consertar" prompts ruins
- ❌ Contradiz gênero (lo-fi com "Neumann pristine"?)

**Conselhos de Iteração:**
- Use com moderação, ou não use
- Não espere milagres
- Melhor focar em estilo e estrutura
- Esta é a técnica de produção mais fraca

---

### 5.5 Keywords "Clean" e "FLAC" para Qualidade

**⚠️ Aviso:** Keywords NÃO aumentam bitrate ou resolução. Podem enviesar Suno para arranjos mais simples, menos distorção, mixing conservador.

**Referência de Keywords:**

| Keyword | Efeito Teórico |
|---------|-----------------|
| `Clean` | Mix balanceado, sem lama |
| `FLAC` | Alta fidelidade, detalhe sonoro |
| `Studio Polish` | Produção profissional |
| `High Fidelity` | Qualidade máxima |
| `Pristine` | Claro, detalhado |
| `Professional Mix` | Bem balanceado |
| `Crystal Clear` | Brilhante, limpo |
| `Mastered` | Som polido final |

**Expectativa Realista:** Melhoria de 5-10% na clareza percebida, no máximo.

**Conselhos de Iteração:**
- Use como tempero, não ingrediente principal
- Não empilhe excessivamente — máximo 2-3 keywords de qualidade
- Combine com gênero — clean funciona para EDM, não para lo-fi
- Combine com boa estrutura — keywords não consertam músicas ruins

## 6. Parâmetros Avançados

### 6.1 Exclude Styles (Excluir Estilos)

**O que faz:** Permite **excluir** elementos específicos que você **NÃO quer** na faixa.

**Disponível para:** Usuários Pro e Premier

**Como Usar:**
- ✅ EM STYLE: "Pop, Female Vocal, Piano, Uplifting"
- ❌ EM EXCLUDE: "Male Vocal, Electric Guitar, Dark"

**Exemplos Práticos:**

**Controle Vocal:**
- Quer apenas voz feminina: Exclua `Male Vocal`
- Quer apenas voz masculina: Exclua `Female Vocal`
- Quer apenas rap: Exclua `Singing`, `Melodic Vocals`

**Controle de Instrumentos:**
- Apenas acústico: Exclua `Electronic`, `Synthesizer`, `Drum Machine`
- Sem piano: Exclua `Piano`, `Keyboard`
- Apenas eletrônico: Exclua `Acoustic Guitar`, `Acoustic Drums`

**Controle de Gênero:**
- Rock puro: Exclua `Electronic`, `Hip Hop`, `Pop`
- Apenas clássico: Exclua `Modern`, `Electronic`, `Pop`

---

### 6.2 Vocal Gender

**O que faz:** Controle direto do gênero da voz principal.

**Opções:** Male | Female

**Combinação com Meta Tags:**
```
VOCAL GENDER: Female
STYLE TAGS: Powerful Vocals, Soulful Vocals, R&B
RESULTADO: Voz feminina poderosa em estilo R&B
```

---

### 6.3 Weirdness Control (Controle de Estranheza)

**O que faz:** Controla quão "criativo" ou "estranho" o resultado pode ser.

**Alcance:** 0% (Seguro) ➜ 100% (Caos) | **50% = Normal**

**Como Usar:**

**🛡️ SEGURO (0-30%): Resultados Previsíveis**
- **Quando usar:** Pop comercial, covers, estilos clássicos
- **Efeito:** SunoAI segue tags fielmente, resultados convencionais
- **Exemplo:** `Pop, Piano Ballad, Female Vocal` → balada padrão

**⚖️ NORMAL (40-60%): Balanceado**
- **Quando usar:** Maioria dos casos
- **Efeito:** Bom equilíbrio entre criatividade e controle
- **Configuração recomendada:** **50%**

**🌪️ CAOS (70-100%): Experimental**
- **Quando usar:** Gêneros avant-garde, fusões incomuns, criatividade máxima
- **Efeito:** Interpretações criativas das tags, resultados imprevisíveis
- **Exemplo:** `Jazz + Electronic` a 90% = fusion muito experimental

**Sugestões de Weirdness:**

| Gênero | % Recomendada |
|--------|---------------|
| Gêneros Clássicos (Pop, Rock) | 30-50% |
| Gêneros Experimentais (Ambient, IDM) | 60-80% |
| Fusões Incomuns | 70-90% |
| Covers/Tributos | 10-30% |

---

### 6.4 Style Influence (Influência de Estilo)

**O que faz:** Controla quão próximo o SunoAI deve seguir suas tags de estilo.

**Alcance:** 0% (Solto) ➜ 100% (Forte)

**Como Usar:**

**🌊 SOLTO (0-30%): Interpretação Livre**
- **Quando usar:** Quer apenas "inspiração" das suas tags
- **Efeito:** SunoAI usa tags como sugestão, muita liberdade criativa
- **Exemplo:** `Rock` a 20% = pode virar soft rock, folk rock, etc.

**⚖️ MODERADO (40-60%): Balanceado**
- **Quando usar:** Quer seguir tags mas com flexibilidade
- **Configuração recomendada:** **50%**

**🔒 FORTE (70-100%): Adesão Estrita**
- **Quando usar:** Tem estilo muito específico em mente
- **Efeito:** SunoAI segue tags muito fielmente
- **Exemplo:** `Progressive Rock, 7/8 time, Synthesizer` a 90% = muito fiel

**Sugestões de Style Influence:**

| Tipo de Tag | % Recomendada |
|-------------|---------------|
| Tags Vagas (Pop, Rock) | 70-90% (para mais controle) |
| Tags Específicas (Progressive Djent Metal) | 40-60% (tags já precisas) |
| Experimentação | 20-40% (para surpresas criativas) |

### 7.7 Melhores Práticas

**Combinando Tags Efetivamente:**
- **Camadas de humor complementares:** `[Dreamy]` + `[Nostalgic]`
- **Balanceie níveis de energia:** Não misture `[High Energy]` com `[Low Energy]` na mesma seção
- **Use estrutura para dinâmica:** Tags diferentes para verso vs refrão
- **Limite total de tags:** 15-20 tags no campo Style
- **Priorize o mais importante primeiro:** Suno dá mais peso às primeiras tags

## 10. Referência Rápida

### 10.1 Checklist de Prompt Completo

**ANTES DE GERAR:**
- [ ] Style: 15-20 tags bem definidas
- [ ] Lyrics: Estrutura clara com seções rotuladas
- [ ] Vocal Gender: Definido (se relevante)
- [ ] Weirdness: Ajustado (30-70% recomendado)
- [ ] Style Influence: Ajustado (50-80% recomendado)
- [ ] Exclude Styles: Elementos indesejados excluídos

---

### 10.2 Tags Mais Efetivas (Alta Confiabilidade)

**★★★★★ Muito Alta:**
- `(parentheses)` para backing vocals
- ALL CAPS para intensidade emocional
- `[Duet]` para duetos

**★★★★☆ Alta:**
- Registros vocais por seção
- Adlibs com colchetes
- Tag stacking com PIPE
- Emoção por seção
- Vocal stretching/stuttering
- Live concert com efeitos sonoros
- Reescrita fonética

**★★☆☆☆ Baixa-Média:**
- Keywords "Clean" e "FLAC"
- Equipment keywords de estúdio

---

### 10.3 Erros Comuns a Evitar

**❌ NÃO FAÇA:**
- Mais de 7 elementos por tag PIPE
- Mais de 4-5 mudanças de registro vocal
- Tudo em ALL CAPS (sem contraste)
- Tags contraditórias na mesma seção
- Mais de 20 tags no Style
- Usar IPA para mais de 1-2 palavras
- Empilhar excessivamente equipment keywords
- Misturar gêneros incompatíveis sem cuidado

**✅ FAÇA:**
- Criar contraste entre seções
- Usar tags complementares
- Priorize elementos mais importantes primeiro
- Combinar emoção com letra
- Testar sistematicamente (3 variações)
- Limitar adlibs e parênteses
- Usar Exclude Styles estrategicamente

---

### 10.10 Dicas Finais Profissionais

**🎯 Combinando Tags Efetivamente:**
- Camadas de humor complementares: `[Dreamy]` + `[Nostalgic]`
- Balanceie níveis de energia: Não misture `[High Energy]` com `[Low Energy]`
- Use estrutura para dinâmica: Tags diferentes para verso vs refrão

**🔄 Progressão de Música:**
- Crie arco claro: Introdução → Desenvolvimento → Clímax → Resolução
- Varie tags entre seções para narrativa
- Use contraste para impacto emocional

**⚡ Estratégias de Geração:**
- Gere 2-3 variações por configuração
- Teste sistematicamente: controle → balanceado → experimental
- Refine baseado nos resultados, não no primeiro generate

## 🎵 Apêndice: Estratégias de Iteração

### Quando um Prompt Falha

**Diagnóstico Rápido:**

| Sintoma | Causa Provável | Solução |
|---------|----------------|---------|
| Vocal ignorado | Tags muito genéricas | Adicione tags específicas de vocal |
| Estrutura confusa | Tags muito longas | Use PIPE, limite a 7 elementos |
| Gênero errado | Style Influence muito baixo | Aumente para 70-80% |
| Muito caótico | Weirdness muito alto | Reduza para 40-50% |
| Muito previsível | Weirdness muito baixo | Aumente para 60-70% |
| Pronúncia errada | Palavra problemática | Use reescrita fonética |
| Sem contraste | Tudo igual | Use ALL CAPS, registros diferentes |
| Duet não funciona | Falta "Duet" no Style | Adicione "Duet" ao Style |

---

### Processo de Refinamento

1. **Primeira Geração:** Use template base
2. **Análise:** Identifique o que funcionou e o que falhou
3. **Ajuste:** Modifique 1-2 elementos por vez
4. **Teste:** Gere 2-3 variações
5. **Compare:** Escolha a melhor versão
6. **Repita:** Até atingir resultado desejado
